для запуска скрипта необходимо:
0. установить python (использовалась версия python 3.10)
1. развернуть базу данных на компьютере
2. установить необходимые зависимости из requirements
3. в файле .env задать значения переменных окружения
(описание переменных в файле .env (обычно он добавляется в gitignore, но это тестовое задание, а не реальный проект))
4. запустить скрипт командой: python main.py
Для корректной работы уведомлений в телеграмм необходимо получить chat_id, он же идентификатор пользователя в @getmyid_bot,
этому пользователю бот будет присылать сообщения.
